# TimeLockedSavings Smart Contract

## Overview

The `TimeLockedSavings` smart contract allows users to deposit Ether with a specified release time. Funds can only be withdrawn after the release time has passed.

## Features

- Time-locked deposits
- Withdrawals only after the release time
- Check deposit details

### Deployment

1. Compile `TimeLockedSavings.sol` using Solidity 0.8.0 or higher.
2. Deploy the contract to an Ethereum network.
